Proceso de votación

El proceso de votación consta de tres pasos:

  1. Acceso a aplicación movil o web de Podemos

    El votante accede a la aplicación movil o web de podemos e introduce su clave de acceso.

  2. Acceso a cabina de votación

    Desde la aplicación o web de Podemos, el votante pincha en un botón que le redirige a la cabina de votación.

  3. Votación

    Una vez validada la clave personal, el votante es redirigido a la cabina de votación. En este momento el votante ha pasado la autenticación y podrá votar.

    1. Rellena la papeleta eligiendo las opciones deseadas para cada pregunta de la votación.
    2. Tras rellenarla, revisa la papeleta pudiendo hacer modificaciones.
    3. Una vez conforme, el sistema cifra y envía la papeleta para su posterior recuento.

Recuento

Tras finalizar el periodo de votación las autoridades proceden a hacer el recuento. Para ello se pondrán de acuerdo para ejecutar, de manera sincronizada, un proceso informático que:

  1. Realizará un barajado de todos los votos emitidos (cifrados), y los descifrará de manera conjunta.
  2. Realizará el recuento a partir de los votos descifrados, según el algoritmo de recuento utilizado.

Verificación

Una vez terminado el recuento se publicará toda la información relativa a la votación necesaria para su validación, tanto por parte de los votantes, como por terceros. Para ellos se publicarán:

  1. Los votos emitidos cifrados, junto con sus localizadores.
  2. Los votos descifrados anonimizados.
  3. Demostraciones matemáticas de la correspondencia entre los votos cifrados (1) y descifrados (2).
  4. Los totales resultantes de aplicar el algoritmo de recuento (plurality y plurality at large) a los votos descifrados.

De esta manera:

  1. Cualquier votante, usando su localizador, podrá certificar que su voto fue debidamente registrado e incluido en el recuento.
  2. Cualquier persona podrá verificar que el conjunto de todos los votos descifrados corresponden con el conjunto de todos los votos emitidos (cifrados).
  3. Cualquier persona podrá verificar que el resultado final del conteo corresponde con una correcta aplicación del algoritmo de recuento a los votos descifrados.

Para facilitar estos tres pasos se proporcionará, mediante descarga, un programa que realiza estas verificaciones de manera automática. El código de este programa será públicamente accesible pudiéndose también auditar.
